Internal Rate of Return (IRR)

Master the concept of IRR – a key financial metric to evaluate investment profitability and make better business decisions.

What is IRR?

The Internal Rate of Return (IRR) is the discount rate at which the Net Present Value (NPV) of all cash flows from a project equals zero. It represents the expected annualized rate of return on an investment.

In simpler terms, IRR is the rate at which an investment breaks even in present value terms. The higher the IRR, the more desirable the investment, assuming all other factors remain the same.

Formula for IRR

There is no direct formula for IRR; it is found using trial and error or software functions (like Excel's =IRR() or =XIRR()). IRR is the rate 'r' that satisfies: 

0 = CF0 + CF1/(1+r) + CF2/(1+r)² + ... + CFn/(1+r)ⁿ

Example

Suppose you invest ₹1,00,000 today and expect to receive ₹40,000 each year for the next 3 years. Using Excel's =IRR() function: 

Year 0: -100000  
Year 1: 40000  
Year 2: 40000  
Year 3: 40000  

=IRR(A1:A4) → Returns ~9.70%

Interpretation

  • If IRR > Required Rate of Return → Accept the project.
  • If IRR < Required Rate of Return → Reject the project.
  • If IRR = Required Rate of Return → Indifferent decision.

Steps to Calculate IRR in Excel

  1. List all cash flows in sequential order, including the initial investment as a negative number.
  2. Select the range of cash flows.
  3. Use the formula =IRR(range) or =XIRR(values, dates) for irregular cash flows.
  4. Compare the IRR with your required rate of return to make a decision.

Advantages of IRR

  • Considers the time value of money.
  • Gives a clear percentage return, easy to interpret.
  • Helps compare multiple projects on a relative basis.

Limitations of IRR

  • Assumes reinvestment at the IRR rate, which may not be realistic.
  • May give multiple IRRs if cash flows change signs multiple times.
  • Does not account for project scale when comparing investments.
